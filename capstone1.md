# Capstone Project on Agentic AI

## Project Title
**CampusAI Lite – Agentic University Information Assistant**

---

## 1. Problem Statement

Develop an Agentic AI application that assists students by answering university-related queries using multiple AI agents, custom tools, and an intelligent workflow.

**Note:** Each student will independently develop a multi-agent AI assistant that answers university-related queries using AI agents, custom tools, workflows, and one or more agentic frameworks.

---

## 2. Mandatory Requirements — Framework Implementation

The project must involve the following frameworks:

- CrewAI
- LangChain
- LangGraph
- AG2 (AutoGen)
- PydanticAI
- Docling
- BeeAI

---

## 3. Tasks

### Part A — Core Implementation

Build **CampusAI Lite** using:

- CrewAI
- LangChain
- LangGraph (or ReAct)

**Required Features:**

- Planner Agent
- Information Agent
- Validation Agent
- One Custom Tool
- Gradio UI

### Part B — Framework Exploration

Students must explore **one** of the following two options using an additional framework:

**Option 1**
Implement the same workflow using AG2 (AutoGen) and compare it with CrewAI.

**Option 2**
Implement BeeAI by building a small proof of concept (e.g., a simple two-agent workflow) and compare its developer experience with CrewAI or AutoGen.
