# TechVista University – Course Catalog 2024-2025

## CS 101 – Introduction to Programming
**Credits:** 3  
**Description:** Foundational programming concepts using Python. Covers variables, control flow, functions, and basic data structures.  
**Prerequisites:** None  
**Offered:** Fall, Spring

## CS 201 – Data Structures & Algorithms
**Credits:** 4  
**Description:** In-depth study of arrays, linked lists, trees, graphs, and algorithmic complexity.  
**Prerequisites:** CS 101  
**Offered:** Fall, Spring, Summer

## CS 310 – Artificial Intelligence
**Credits:** 3  
**Description:** Introduction to AI including search, logic, machine learning, and natural language processing.  
**Prerequisites:** CS 201, MATH 202  
**Offered:** Fall, Spring

## CS 450 – Deep Learning
**Credits:** 3  
**Description:** Neural network architectures including CNNs, RNNs, and Transformers. Practical assignments using PyTorch.  
**Prerequisites:** CS 310, MATH 305  
**Offered:** Fall

## DS 201 – Statistical Foundations
**Credits:** 3  
**Description:** Probability, statistics, hypothesis testing, and regression analysis.  
**Prerequisites:** MATH 101  
**Offered:** Fall, Spring

## DS 301 – Machine Learning
**Credits:** 4  
**Description:** Supervised, unsupervised, and reinforcement learning algorithms.  
**Prerequisites:** DS 201, CS 201  
**Offered:** Fall, Spring

## AI 501 – Large Language Models
**Credits:** 3  
**Description:** Architecture, training, fine-tuning, and deployment of large language models. Includes hands-on work with open-source models.  
**Prerequisites:** CS 450  
**Offered:** Spring

## MBA 601 – Business Analytics
**Credits:** 3  
**Description:** Data-driven decision making, business intelligence tools, and KPI frameworks.  
**Prerequisites:** Admission to MBA program  
**Offered:** Fall, Spring
